ST-LINK flash and debug on the SPRacingF3

I was having trouble flashing BetaFlight to my SPRacingF3 board, so I used the SWD port to load the firmware.

In one terminal, run:

openocd -f interface/stlink-v2.cfg -f target/stm32f3x_stlink.cfg

In another, run:

telnet localhost 4444
reset halt
sleep 100
poll
flash probe 0
flash write_image erase obj/betaflight_SPRACINGF3.hex
sleep 200
poll
reset run

The output will look like:

system in cleanflight/ on betaflight
› telnet localhost 4444
Trying ::1...
telnet: connect to address ::1: Connection...

Naze32 ST-Link Debugging

This is a quick guide to building and flashing CleanFlight via ST-Link

To build a custom version of CleanFlight for the NAZE 32:

cd into your cleanflight source directory (mine is in ~/src/rc/cleanflight) and run:

TARGET=NAZE make clean && TARGET=NAZE make

in one terminal, run:

openocd -f support/openocd/stm32f10x.cfg

where support/openocd/stm32f10x.cfg:

source [find interface/stlink-v2.cfg]
source [find target/stm32f1x.cfg]
# if you have the reset line plugged in:
#reset_config srst_only...

Building Android on Ubuntu 14.04

Install Ubuntu 14.04 and configure the required packages

# update packages
sudo apt-get update -y

# install Java
sudo apt-get install openjdk-7-jdk

# cleanup
sudo apt-get autoremove -y
sudo apt-get update -y
sudo apt-get dist-upgrade -y
sudo dpkg --configure -a

# install deps
#  https://code.google.com/p/android/issues/detail?id=178284
sudo apt-get install -y g++-multilib
sudo apt-get install -y bison g++-multilib git gperf libxml2-utils make python-networkx zlib1g-dev:i386 zip

Decompiling an APK

Grab apktool

brew install apktool

Download an APK or copy one off your (rooted) phone

Then decompile the apk

apktool d my-cool.apk

Open up the smali source folder and edit the sources

To recompile the apk after making changes

apktool b my-cool.apk

apktool makes it insanely easy to decompile an apk

To edit in java

brew install dex2jar
d2j-dex2jar my-cool.apk

Download JD-GUI

Open jd-gui.app and then pick my-cool.apk.jar
